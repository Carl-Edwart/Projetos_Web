## 📂 Organização dos Trabalhos

Os projetos estão estruturados de forma sequencial, permitindo uma análise clara da progressão de conhecimento:

```
/
├── Trabalho-01/
├── Trabalho-02/
├── Trabalho-03/
├── ...
└── README.md
```

Cada diretório contém:

* Código-fonte do projeto
* Arquivo `README.md` específico
* Recursos auxiliares (imagens, scripts, estilos, etc.)

A numeração segue a ordem cronológica das atividades propostas.

## 🧰 Tecnologias Utilizadas

Ao longo dos trabalhos, são exploradas as seguintes tecnologias:

* **HTML5** — Estrutura e semântica

## 🚀 Execução dos Projetos

Cada trabalho pode ser executado de forma independente:

1. Acesse o diretório do projeto desejado
2. Abra o arquivo `index.html` no navegador
3. Não há necessidade de instalação adicional
