# Site Pessoal – Projeto PW III

## 📌 Sobre o projeto
Este projeto foi desenvolvido como atividade acadêmica da disciplina **Programação Web III (PW III)**, ministrada pelo professor **André Olímpio**.

O principal objetivo do trabalho é demonstrar nossas habilidades em **HTML**, já integrando **conhecimentos técnicos utilizados no mercado**, como:
- Uso de repositório GitHub
- Arquivos `LICENSE`, `.gitignore` e `README`
- Organização profissional de pastas e código

Além do aspecto técnico, o site tem como tema a **minha trajetória acadêmica**, apresentando informações pessoais, evolução nos estudos e interesses na área de tecnologia.

---

## 🎯 Objetivo
- Aplicar conceitos de HTML de forma estruturada  
- Utilizar CSS e JavaScript para estilização e interatividade  
- Seguir boas práticas de versionamento e organização de projetos  
- Simular um projeto com padrão próximo ao ambiente profissional  

---

## 🛠️ Tecnologias utilizadas
- **HTML5** – Estrutura do site  
- **CSS3** – Estilização e layout  
- **JavaScript** – Interatividade e comportamento dinâmico
