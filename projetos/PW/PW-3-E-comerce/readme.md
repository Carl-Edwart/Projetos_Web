# PW-3: E-commerce - Sistema de Carrinho de Compras

## 📋 Descrição
Projeto desenvolvido para a disciplina de Programação Web III. Sistema web simples de carrinho de compras construído com HTML, CSS e JavaScript puro.

## ✨ Funcionalidades
- ✅ Listagem de produtos
- ✅ Adicionar produtos ao carrinho
- ✅ Remover produtos do carrinho
- ✅ Atualizar quantidade de itens
- ✅ Calcular total do carrinho
- ✅ Persistência de dados com localStorage
- ✅ Filtro de produtos por faixa de preço (usando switch)

## 🛠️ Tecnologias
- **HTML5**: Estrutura semântica
- **CSS3**: Estilização e design responsivo
- **JavaScript (ES6+)**: Lógica pura, sem frameworks

## 📂 Estrutura do Projeto
```
PW-3-E-comerce/
├── index.html              # Página principal da loja
├── pages/
│   └── carrinho.html       # Página do carrinho
├── styles/
│   ├── main.css            # Estilos globais
│   ├── index.css           # Estilos da página inicial
│   └── carrinho.css        # Estilos do carrinho
├── scripts/
│   ├── main.js             # Inicialização
│   ├── produtos.js         # Gerenciamento de produtos
│   ├── carrinho.js         # Operações do carrinho
│   ├── storage.js          # Persistência com localStorage
│   ├── filtros.js          # Filtros de produtos
│   └── utils.js            # Funções utilitárias
└── assets/
    ├── images/             # Imagens do projeto
    └── icons/              # Ícones
```

## 🚀 Como Usar
1. Clone ou baixe o projeto
2. Abra o arquivo `index.html` no navegador
3. Navegue pela loja e use o carrinho

## 📝 Requisitos Técnicos Implementados
- ✓ Array e Object para produtos e carrinho
- ✓ Funções bem definidas e organizadas
- ✓ Manipulação de DOM (querySelector, createElement, appendChild, addEventListener)
- ✓ Estruturas: if/else, for/forEach, switch
- ✓ localStorage para persistência
